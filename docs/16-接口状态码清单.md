# 课堂 Vibe Coding 平台接口状态码清单

## 1. 文档目标

本清单用于统一平台接口返回的业务状态码，便于：

- 前后端联调
- 教师端与学生端错误提示统一
- 日志检索与告警治理
- 自动评分、公告发布、配置面板等模块共享同一错误语义

## 2. 设计原则

- 状态码按模块分段
- 同一错误语义只保留一个主码
- 面向前端可理解，面向后端可追踪
- 首期先覆盖核心模块，避免过度细分

## 3. 通用规则

| 范围 | 含义 |
| --- | --- |
| 0 | 成功 |
| 4001xx | 通用请求与参数错误 |
| 4002xx | 权限与访问错误 |
| 4003xx | Schema / Patch 相关错误 |
| 4004xx | 项目 / 资源不存在或状态冲突 |
| 4005xx | 公告展示与发布错误 |
| 5001xx | 自动评分与认知分析错误 |
| 5002xx | 代码生成与联动校验错误 |
| 5003xx | 运行调度与容器错误 |
| 5009xx | 未分类内部错误 |

## 14. 前端提示等级建议

| 状态码范围 | 等级 | 建议处理方式 |
| --- | --- | --- |
| 4001xx | warning | 表单或请求参数提示 |
| 4002xx | error | 显示无权限提示并阻止继续 |
| 4003xx | warning / error | 显示配置冲突与修复建议 |
| 4004xx | error | 提示资源不存在或状态异常 |
| 4005xx | warning | 引导教师调整公告配置 |
| 5001xx | warning | 允许稍后重试 |
| 5002xx | error | 触发修复流程或人工确认 |
| 5003xx | error | 提示资源或运行环境异常 |
| 5009xx | error | 显示通用系统错误 |

## 15. 日志字段建议

每次错误建议统一记录：

- requestId
- code
- message
- module
- actorType
- actorId
- projectId
- courseId
- classId
- createdAt
