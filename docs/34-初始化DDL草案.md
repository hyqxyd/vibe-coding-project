# 课堂 Vibe Coding 平台初始化 DDL 草案

## 1. 文档目标

本草案用于把前面的数据库表设计草案与字段字典进一步收敛成可执行的初始化建表脚本方向，便于后续：

- 建立首期开发库
- 生成迁移脚本
- 对齐 ORM 实体设计
- 校验索引与约束是否覆盖核心链路

## 2. 适用范围

本草案面向首期 MVP 在线业务库，优先覆盖：

- 组织域
- 教学域
- 项目域
- 过程域
- 评分与公告域
- 平台治理域

## 3. 方言选择说明

首期建议以 PostgreSQL 为目标数据库，原因是：

- 与前期平台技术选型保持一致
- 适合结构化业务数据与 JSONB 混合存储
- 便于后续索引、统计与归档扩展

本草案中逻辑层的 `datetime`、`tinyint`、`json` 在 DDL 中统一映射为：

- `datetime` -> `timestamptz`
- `tinyint` -> `boolean`
- `json` -> `jsonb`

## 5. 初始化 DDL 草案

```sql
create table if not exists app_user (
    id bigint generated by default as identity primary key,
    user_code varchar(64) not null unique,
    name varchar(100) not null,
    role_type varchar(32) not null,
    status varchar(32) not null default 'active',
    created_at timestamptz not null default now(),
    updated_at timestamptz not null default now()
);

create table if not exists course (
    id bigint generated by default as identity primary key,
    course_code varchar(64) not null unique,
    course_name varchar(200) not null,
    teacher_id bigint references app_user(id),
    status varchar(32) not null default 'active',
    created_at timestamptz not null default now(),
    updated_at timestamptz not null default now()
);

create table if not exists class (
    id bigint generated by default as identity primary key,
    course_id bigint not null references course(id),
    class_code varchar(64) not null unique,
    class_name varchar(200) not null,
    teacher_id bigint references app_user(id),
    created_at timestamptz not null default now(),
    updated_at timestamptz not null default now()
);

create table if not exists class_user (
    id bigint generated by default as identity primary key,
    class_id bigint not null references class(id),
    user_id bigint not null references app_user(id),
    role_in_class varchar(32) not null,
    created_at timestamptz not null default now(),
    unique (class_id, user_id)
);

create table if not exists class_session (
    id bigint generated by default as identity primary key,
    class_id bigint not null references class(id),
    session_code varchar(64) not null unique,
    session_name varchar(200) not null,
    start_time timestamptz,
    end_time timestamptz,
    status varchar(32) not null default 'draft',
    created_at timestamptz not null default now()
);

create table if not exists topic (
    id bigint generated by default as identity primary key,
    session_id bigint not null references class_session(id),
    topic_code varchar(64) not null unique,
    topic_name varchar(200) not null,
    description text,
    sort_order integer not null default 0,
    enabled boolean not null default true
);

create table if not exists rubric_profile (
    id bigint generated by default as identity primary key,
    course_id bigint not null references course(id),
    rubric_code varchar(64) not null unique,
    name varchar(200) not null,
    weights_json jsonb not null,
    dimensions_json jsonb not null,
    status varchar(32) not null default 'active',
    created_at timestamptz not null default now()
);

create table if not exists question (
    id bigint generated by default as identity primary key,
    topic_id bigint not null references topic(id),
    question_code varchar(64) not null unique,
    question_title varchar(200) not null,
    description text,
    rubric_id bigint references rubric_profile(id),
    public_show_enabled boolean not null default false,
    created_at timestamptz not null default now()
);

create table if not exists class_constraint (
    id bigint generated by default as identity primary key,
    session_id bigint not null references class_session(id),
    allow_free_exploration boolean not null default true,
    allow_dependency_install boolean not null default false,
    public_show_enabled boolean not null default true,
    tech_stack_profile varchar(100) not null,
    updated_by bigint references app_user(id),
    updated_at timestamptz not null default now(),
    unique (session_id)
);

create table if not exists student_project (
    id bigint generated by default as identity primary key,
    project_code varchar(64) not null unique,
    student_id bigint not null references app_user(id),
    class_id bigint not null references class(id),
    session_id bigint not null references class_session(id),
    topic_id bigint not null references topic(id),
    question_id bigint not null references question(id),
    title varchar(200) not null,
    description text,
    status varchar(32) not null default 'draft',
    tech_stack_profile varchar(100) not null,
    current_snapshot_id bigint,
    created_at timestamptz not null default now(),
    updated_at timestamptz not null default now()
);

create table if not exists project_schema_snapshot (
    id bigint generated by default as identity primary key,
    project_id bigint not null references student_project(id),
    snapshot_code varchar(64) not null unique,
    schema_version varchar(32) not null,
    snapshot_content jsonb not null,
    summary varchar(500),
    created_by_type varchar(32) not null,
    created_by_id bigint references app_user(id),
    created_at timestamptz not null default now()
);

do $$
begin
    if not exists (
        select 1
        from pg_constraint
        where conname = 'fk_student_project_current_snapshot'
    ) then
        alter table student_project
            add constraint fk_student_project_current_snapshot
            foreign key (current_snapshot_id) references project_schema_snapshot(id);
    end if;
end $$;

create table if not exists project_patch (
    id bigint generated by default as identity primary key,
    project_id bigint not null references student_project(id),
    patch_code varchar(64) not null unique,
    base_snapshot_id bigint not null references project_schema_snapshot(id),
    source varchar(32) not null,
    actor_type varchar(32) not null,
    actor_id bigint references app_user(id),
    reason varchar(500),
    operations_json jsonb not null,
    apply_status varchar(32) not null default 'pending',
    created_at timestamptz not null default now()
);

create table if not exists project_file_change (
    id bigint generated by default as identity primary key,
    project_id bigint not null references student_project(id),
    patch_id bigint not null references project_patch(id),
    file_path varchar(500) not null,
    file_domain varchar(64) not null,
    action varchar(32) not null,
    created_at timestamptz not null default now()
);

create table if not exists project_event (
    id bigint generated by default as identity primary key,
    event_code varchar(64) not null unique,
    project_id bigint not null references student_project(id),
    session_id bigint not null references class_session(id),
    event_type varchar(64) not null,
    stage_tag varchar(64),
    actor_type varchar(32) not null,
    actor_id bigint references app_user(id),
    payload_json jsonb not null,
    created_at timestamptz not null default now()
);

create table if not exists project_conversation (
    id bigint generated by default as identity primary key,
    project_id bigint not null references student_project(id),
    message_role varchar(32) not null,
    content text not null,
    summary varchar(500),
    stage_tag varchar(64),
    created_at timestamptz not null default now()
);

create table if not exists project_run_record (
    id bigint generated by default as identity primary key,
    project_id bigint not null references student_project(id),
    run_no integer not null,
    status varchar(32) not null,
    sandbox_id varchar(128),
    preview_url varchar(500),
    log_summary text,
    started_at timestamptz,
    ended_at timestamptz
);

create table if not exists cognition_profile (
    id bigint generated by default as identity primary key,
    project_id bigint not null references student_project(id),
    current_stage varchar(64) not null,
    ai_dependency_level varchar(32) not null,
    iteration_count integer not null default 0,
    failed_run_count integer not null default 0,
    recovered_run_count integer not null default 0,
    metrics_json jsonb not null default '{}'::jsonb,
    flags_json jsonb not null default '[]'::jsonb,
    updated_at timestamptz not null default now(),
    unique (project_id)
);

create table if not exists teacher_score (
    id bigint generated by default as identity primary key,
    project_id bigint not null references student_project(id),
    rubric_id bigint not null references rubric_profile(id),
    process_score numeric(5,2),
    result_score numeric(5,2),
    final_score numeric(5,2),
    dimensions_json jsonb,
    comment text,
    teacher_id bigint not null references app_user(id),
    created_at timestamptz not null default now(),
    updated_at timestamptz not null default now()
);

create table if not exists auto_score_result (
    id bigint generated by default as identity primary key,
    project_id bigint not null references student_project(id),
    rubric_id bigint not null references rubric_profile(id),
    process_score numeric(5,2),
    result_score numeric(5,2),
    final_score numeric(5,2),
    evidence_json jsonb,
    risk_flags_json jsonb,
    status varchar(32) not null default 'pending',
    generated_at timestamptz
);

create table if not exists showcase_record (
    id bigint generated by default as identity primary key,
    project_id bigint not null references student_project(id),
    title varchar(200) not null,
    summary text,
    public_fields_json jsonb not null,
    result_url varchar(500),
    expires_at timestamptz,
    published_by bigint not null references app_user(id),
    status varchar(32) not null default 'published',
    created_at timestamptz not null default now()
);

create table if not exists resource_quota_log (
    id bigint generated by default as identity primary key,
    session_id bigint not null references class_session(id),
    cpu varchar(32) not null,
    memory varchar(32) not null,
    ttl_minutes integer not null,
    updated_by bigint not null references app_user(id),
    created_at timestamptz not null default now()
);

create table if not exists audit_log (
    id bigint generated by default as identity primary key,
    actor_type varchar(32) not null,
    actor_id bigint references app_user(id),
    action_type varchar(64) not null,
    target_type varchar(64) not null,
    target_id bigint not null,
    detail_json jsonb not null,
    created_at timestamptz not null default now()
);
```

## 6. 索引草案

```sql
create index if not exists idx_class_course_id on class(course_id);
create index if not exists idx_class_user_class_id on class_user(class_id);
create index if not exists idx_class_user_user_id on class_user(user_id);
create index if not exists idx_class_session_class_id on class_session(class_id);
create index if not exists idx_topic_session_id on topic(session_id);
create index if not exists idx_question_topic_id on question(topic_id);
create index if not exists idx_student_project_student on student_project(student_id, status);
create index if not exists idx_student_project_class_question on student_project(class_id, question_id, status);
create index if not exists idx_project_snapshot_project on project_schema_snapshot(project_id, created_at desc);
create index if not exists idx_project_patch_project on project_patch(project_id, created_at desc);
create index if not exists idx_project_patch_status on project_patch(apply_status, created_at desc);
create index if not exists idx_project_file_change_project on project_file_change(project_id, created_at desc);
create index if not exists idx_project_event_project on project_event(project_id, created_at desc);
create index if not exists idx_project_event_session_type on project_event(session_id, event_type, created_at desc);
create index if not exists idx_project_conversation_project on project_conversation(project_id, created_at desc);
create index if not exists idx_project_run_record_project on project_run_record(project_id, started_at desc);
create index if not exists idx_teacher_score_project on teacher_score(project_id);
create index if not exists idx_auto_score_project_status on auto_score_result(project_id, status);
create index if not exists idx_showcase_status_expires on showcase_record(status, expires_at);
create index if not exists idx_resource_quota_session on resource_quota_log(session_id, created_at desc);
create index if not exists idx_audit_target on audit_log(target_type, target_id, created_at desc);
```

## 7. 初始数据建议

首期建议初始化以下基础数据：

- 管理员账号
- 教师测试账号
- 学生测试账号
- 默认技术栈 profile
- 默认课程级 Rubric 模板
- 默认课堂约束配置

## 8. 迁移拆分建议

建议后续把初始化脚本拆成多个迁移版本：

1. 组织与教学基础表
2. 项目与 Schema 表
3. 过程留痕表
4. 评分与公告表
5. 平台治理与索引

## 9. 注意事项

- `student_project.current_snapshot_id` 采用后置外键，避免循环依赖导致建表失败
- 高频写入表优先保证主键、外键和核心查询索引，不建议首期建立过多复合索引
- `project_conversation`、`project_event`、`project_run_record` 需要为后续归档迁移保留结构一致性
- 自动评分与认知分析相关 JSON 字段首期先保留弹性，后续再视查询需求拆表
