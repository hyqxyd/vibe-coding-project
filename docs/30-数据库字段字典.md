# 课堂 Vibe Coding 平台数据库字段字典

## 1. 文档目标

本字典用于补充数据库表设计草案，对首期关键表的字段含义、类型、约束与备注进行统一说明，方便：

- 后端建表
- ORM / 实体定义
- 接口字段映射
- 数据迁移与归档

## 2. 覆盖范围

首期优先覆盖以下核心表：

- user
- course
- class
- class_session
- topic
- question
- student_project
- project_schema_snapshot
- project_patch
- project_event
- project_conversation
- project_run_record
- rubric_profile
- teacher_score
- auto_score_result
- showcase_record

## 3. user

| 字段 | 类型 | 约束 | 说明 |
| --- | --- | --- | --- |
| id | bigint | PK | 主键 |
| user_code | varchar(64) | unique, not null | 用户编码 |
| name | varchar(100) | not null | 用户姓名 |
| role_type | varchar(32) | not null | student / teacher / admin |
| status | varchar(32) | not null | active / disabled |
| created_at | datetime | not null | 创建时间 |
| updated_at | datetime | not null | 更新时间 |
