# 课堂 Vibe Coding 平台教师端接口联调用例表

## 1. 文档目标

本用例表用于把教师端接口契约和联调清单落成可执行的逐条联调用例，方便：

- 前后端联调执行
- 测试回归
- 问题定位与复测

## 2. 使用说明

每条用例建议至少记录：

- 用例编号
- 目标接口
- 前置条件
- 输入参数
- 期望响应
- 页面期望表现
- 实际结果

## 3. 课程总览接口用例

| 用例编号 | 接口 | 场景 | 前置条件 | 期望结果 |
| --- | --- | --- | --- | --- |
| T-OV-001 | GET /overview | 正常加载课程总览 | 存在课程与班级 | 返回 summary 与 charts，页面指标卡正常渲染 |
| T-OV-002 | GET /overview | 按 topicId 筛选 | 存在多个主题 | 返回指定主题数据 |
| T-OV-003 | GET /overview | 无权限访问 | 当前教师不属于该课程 | 返回 400202 或 400203 |

## 11. 页面表现验收要点

联调时除接口外，还需确认：

- 空态是否正确
- 加载态是否正确
- 错误态是否显示匹配文案
- 排序与筛选是否同步到页面
- 提交成功后页面是否刷新
